Automating Tasks with Ansible

Posted on June 25, 2025

---
- name: Install and start httpd
  hosts: webservers
  become: yes
  tasks:
    - name: Install httpd
      yum:
        name: httpd
        state: latest
    - name: Start httpd service
      service:
        name: httpd
        state: started
        enabled: yes
